The following compiler implementations are included with the Shared Source CLI (SSCLI) 2.0 and C# 2.0 distribution.

Compiler Description Source location
C# 2.0 compiler The SSCLI 2.0 implementation of the C# 2.0 compiler. The code for this compiler is written in C++. sscli20\csharp
JScript compiler The SSCLI 2.0 implementation of the JScript compiler. The code for this compiler is written in C#. sscli20\jscript
