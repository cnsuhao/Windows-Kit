Make Utility (nmake.exe)

This Shared Source CLI build tool is a port of the Microsoft Program Maintenance Utility that is included with Microsoft® Visual C++® 2005. The application binary file is named nmake.exe on Microsoft Windows®.

This tool is documented in the Visual C++® 2005 documentation or in the Microsoft MSDN® documentation:

http://msdn2.microsoft.com/library/dd9y37ha(en-US,VS.80).aspx

The source code for this tool can be found in the sscli20\tools\nmake\ directory.
