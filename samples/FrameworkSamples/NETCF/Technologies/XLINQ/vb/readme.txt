Summary description 
--------------------
This sample highlights the key features of XLINQ. This sample is a simple Books application that loads a list of books from an xml file and lets the user do the following operations
1)Search by book title 
2)View the list of books grouped by Author, Style or Subject
3)Do a unified search, where the user can search by any of the book's attributes. 

This sample also saves the last 5 unified search queries done by the user to an Application config file. 


Build requirements
------------------
1. Visual Studio 2008 (Orcas) with .NET Compact Framework installed.



How to build
------------
Open the solution with Visual Studio 2008 and run the project.
